TECNOLOGÍAS Y DEPENDENCIAS:
===========================
- Java 21
- Spring Boot 4.0.2
- Spring Data JPA (para ORM y persistencia)
- Spring Security (para autenticación y autorización)
- Spring OAuth2 Resource Server (para JWT)
- Spring Validation (para validación de datos)
- Spring Web MVC (para REST API)
- PostgreSQL (base de datos principal - Supabase)
- H2 Database (base de datos en memoria para desarrollo)
- Lombok (para reducir código boilerplate)
- JSON Web Token (JWT) - jjwt 0.11.5
